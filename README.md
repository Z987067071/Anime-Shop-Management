# Anime Shop Management后台系统

#### 观前提示

本项目大部分均由AI辅助开发,主要功能基本已经拥有,但是部分功能还是有没完善的地方,如果有什么好的设想可以指出一下,本系统用于毕业设计项目,论文结束后可能停止进行开发,有需要对这个系统进行参考的,随时欢迎来参考以及互相学习交流.

### 环境

Java 17.0.17

IDEA 2025.2.5

Redis-x64-5.0.14.1

### 功能介绍

后台系统基于Springboot进行开发

平台目前包括两部分:

- 移动端: 首页,商品,搜索,排行榜,分类,购物车,订单,个人中心,商品收藏,票夹,提交工单,社区.
- 后台: 账户管理,商品管理,工单管理,评论管理,分类管理

该仓库为后端代码,还请结合前端仓库代码一起启动

### 部署运行

#### 前端运行步骤

- 首先拉取代码

```
git clone https://github.com/Z987067071/Anime-Shop-Management.git
```

- 安装好对应的Java版本,将目录里的anime_shop.sql放进数据库 
- 解压好Redis后在其文件夹里面打开命令行cmd, 运行指令为 redis-server.exe redis.windows.conf
- 用IDEA打开项目,由于是本地部署所以说文件上传都会在后端的静态资源文件夹里,resources里面有头像上传和评论图片帖子图片,工单图片等,**你可以将里面的多余的jpg全部删掉因为是我之前测试功能上传的懒得删了**
- **上传图片可能会存在问题需要你去FileUploadUtil文件修改存放路径以及WebMvcConfig(非常重要!!!)**
- 右键目录文件pom.xml然后Maven去Sync project,如果没什么反应你就去修改下项目结构的maven目录什么的,或者在build里找到reload all projects重新装,**至少我之前遇到的问题是这么解决的**
- 然后你就可以开始尝试运行了
